Keyboard shortcuts

Keyboard navigation follows the filter form and then each task column. On a focused task card, use Left or Right Arrow to move the task between columns.

  • Alt + Shift + D — Open the dashboard
  • Alt + Shift + S — Open settings
  • Alt + Shift + F — Focus filters
  • Alt + Shift + T — Focus the task board
  • Alt + Shift + K — Focus this shortcut help
  • Left/Right Arrow — Move a focused task to the previous or next column
